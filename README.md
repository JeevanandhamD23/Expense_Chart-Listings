## Run Locally

Clone the project

```bash
  git clone https://github.com/JeevanandhamD23/Expense_Chart-Listings
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm Start
```
